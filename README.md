# Kinexica SpoilSense-Broker

SpoilSense-Broker is an autonomous negotiation system and dashboard for perishable goods.

## Structure

- `data_pipeline/`: Edge sensor simulation
- `pinn_engine/`: Chemical thermodynamic modeling
- `agent_broker/`: Multi-agent negotiation system
- `dashboard/`: UI for the judges
- `main.py`: Orchestrator tying it all together
